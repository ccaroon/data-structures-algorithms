# Factorial
Factorial is denoted by an integer followed by `!`, e.g. `42!`.

`n! = n * (n-1) * (n-2) * (n-3) * ... 3 * 2 * 1`

Example: `5! = 5 * 4 * 3 * 2 * 1 = 120`

By Convention `0! = 1`.
